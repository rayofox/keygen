# wingide keygen
*目前支持到版本11.x.x*
```
Usage : python3 keygen_wingide.py [action] [param]  
action : g|gen|t|test  

g|gen  : generate the activation code, default action is gen  
python3 keygen_wingide.py  
python3 keygen_wingide.py gen  
python3 keygen_wingide.py g  

t|test : test the new version magic , require 3 codes from another crask tool such as DVT or purchase method!  
python3 keygen_wingide.py t    licenseCode requestCode  activationCode  
python3 keygen_wingide.py test licenseCode requestCode  activationCode  
```
